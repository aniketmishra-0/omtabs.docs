OmTabs Privacy Policy (Free Mode)

This document applies to the current OmTabs build where all features are free and premium activation/payment flow is disabled.

All features free Local-first storage No hidden tracking No account required
Home Privacy Free Privacy Support Terms

What OmTabs Accesses

  • Tab title and tab URL for rename and restore workflows.
  • Pinned state and tab-group metadata for group management tools.
  • Only normal web tabs (http and https).

What OmTabs Stores

  • Rename memory, URL rules, group rules, and workspace snapshots.
  • Session backups, undo/redo timeline, theme settings, and sync preferences.
  • Optional privacy lock secrets as salted hashes (not plain text).
  • By default, data remains in browser-local extension storage.

Free-Mode Billing and Premium Note

  • No premium unlock code is required in this build.
  • No payment step is required to use workspaces, config backup, sync, or privacy tools.
  • No billing data is collected by OmTabs for this build mode.

What OmTabs Does Not Do

  • Does not sell personal data.
  • Does not send browsing history to OMDOCS servers.
  • Does not require account signup for extension usage.
  • Does not include mandatory third-party analytics for core functions.

Permissions Summary

  • tabs: read/update tab state.
  • storage: save extension settings.
  • scripting: apply title overrides on supported pages.
  • contextMenus: right-click rename action.
  • tabGroups: manage tab groups.
  • management: compatibility checks in extension context.
