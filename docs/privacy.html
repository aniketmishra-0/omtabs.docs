OmTabs Privacy Policy

OmTabs is developed by OMDOCS. This page explains what data the extension accesses and how it is used.

Local-first storage No data sale No hidden tracking User-controlled locks
Home Free Privacy Privacy Support Terms

What OmTabs Accesses

  • Tab title and URL.
  • Tab grouping state and pinned state.
  • Only on normal web pages (http and https).

What OmTabs Stores

  • Rename settings, URL memory, and rule configuration.
  • Workspace snapshots, backup history, and optional sync preferences.
  • Optional lock credentials (stored as salted hashes, not plain text).
  • By default data remains in local browser profile storage.

What OmTabs Does Not Do

  • Does not sell personal data.
  • Does not include third-party analytics for core workflows.
  • Does not require user account for base usage.
  • Does not send browsing history to OMDOCS servers.

Permissions Summary

  • tabs: read/update tab state.
  • storage: save extension settings.
  • scripting: apply title overrides on supported pages.
  • contextMenus: right-click rename action.
  • tabGroups: tab group management.
  • management: extension mode checks and compatibility handling.
