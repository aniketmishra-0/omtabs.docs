Before Contacting Support

  1. Confirm extension version from chrome://extensions.
  2. Capture screenshot of the popup error/status.
  3. If sync or restore issue appears, capture exact steps and browser version.

For Feature Access Issues

  • Feature name (workspace, backup, sync, privacy lock, etc.)
  • Error message shown in popup
  • Date/time of issue
  • Browser + OS details
  • Error screenshot from extension popup
